How do you create a new git repository locally?
Create a directory, go into it and use 'git init' to start tracking changes. Otherwise, you can use 'git clone' to take a repository from github.com and copy it to a local folder.
How would you find out what files are currently staged?
Type 'git status'. All of the staged files will appear green.
